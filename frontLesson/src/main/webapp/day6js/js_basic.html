자바스크립트 데이터 타입

참고

JavaScript has 8 Datatypes

1. String : 문자열 리터럴은 '' 또는 "" 기호 사용
2. Number : 정수 또는 실수
3. Bigint
4. Boolean : true, false
5. Undefined : 자바스크립트 특징. 값이 없는 변수는 undefined(타입,값)
6. Null
8. Object : 객체.(배열 포함)
