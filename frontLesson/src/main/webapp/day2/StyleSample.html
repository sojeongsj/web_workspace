CSS

HTML이 콘텐츠의 구조와 의미를 정의하는 반면 CSS는 스타일과 레이아웃을 지정합니다. 예를 들어, CSS를 사용하면 콘텐츠의 글꼴, 색상, 크기 및 간격을 변경하거나, 여러 개의 열로 분할하거나, 애니메이션이나 기타 장식 효과를 추가할 수 있습니다.
Cascading Stylesheets (종속형 스타일시트)
- HTML이 콘텐츠의 구조와 의미를 정의하는 반면 CSS는 스타일과 레이아웃을 지정합니다. 예를 들어, CSS를 사용하면 콘텐츠의 글꼴, 색상, 크기 및 간격을 변경하거나, 여러 개의 열로 분할하거나, 애니메이션이나 기타 장식 효과를 추가할 수 있습니다.

- 태그의 style 속성으로 설정하거나 head 태그 안에서 style 태그로 작성합니다. - style 태그는 어떤 요소에 적용할 것인지 선택자(selector)를 사용합니다.

참고링크
a 는 anchor 의 약자입니다. 링크에는 href 속성값에 이동할 웹페이지의 주소를 지정합니다.이것은 URL(Uniform Resource Locator) 이라고 합니다. 웹상의 어디에 위치하는지 주소입니다. 예를 들면, Mozilla's 영어 홈페이지는 https://www.mozilla.org 모질라
url 은 절대 URL 또는 상대 URL 로 나타냅니다. 절대 URL은 프로토콜과 도메인네임 예를 들면 https://www.mozilla.org 부터 모두 표시하고 , 상대 URL은 도메인네임이 같은 경우 생략하고 표시합니다. 예시로 절대주소는 <a href="http://localhost:8081/index.html">링크 </a> 이고 상대주소는 <a href="index.html">링크 </a> 입니다. url은 img 태그의 src 속성에도 동일하게 적용합니다.
