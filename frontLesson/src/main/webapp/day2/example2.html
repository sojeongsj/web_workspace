CSS 프로퍼티

번호 속성 설명 예시
1 width, hieght 가로길이, 세로길이 width : 100px; height : 80%;
2 margin, padding 바깥 여백, 안쪽 여백 margin : 20px; -상하좌우 20px
margin : 10px 20px; -상하 10px, 좌우 20px
padding : 10px 20px 30px 40px;
-상 10px, 우 20px, 하 30px, 좌 40px
3 display 요소를 어떻게 보여줄지 결정
  • none : 안보임
  • block : 블록박스
  • inline : 인라인박스
  • inline-block : block과 inline의 중간 형태
 <style> .display-none{ display: none } </style>
<div class="display-none">1</div>
4 color 글자의 색상
  • red, blue 와 같이 정의된 색
  • #ffffff, #000 와 같은 16진수 색상코드
  • rgb(255,255,255) 와 같은 rgb
  • rgba(200, 100, 150, 0.5) 와 같은 투명도가 적용된 rgba
 color : red;
color : #ffffff;
color : rgb(255,255,255);
5 font 글자의 폰트를 정의하는 속성
  • font-style : 이탤릭체 등 글꼴의 스타일 지정 - normal, italic
  • font-weight : 글꼴의 두께 - lighter, normal, bold, bolder
  • font-size : 글짜 크기 - px 등 단위, small, big
  • font-family : 쉼표로 여러글꼴 등록, 맨앞 글꼴 우선 적용, 맨앞 글꼴이 사용자의 컴퓨터에 없을 시 다음 글꼴을 적용
  • font-variant : 글꼴 변형
  • font-height : 줄 간격
 .box1 { font-size: 20px; font-family: 나눔고딕,NanumGothic,돋움,Dotum; }
.ex2 { font: italic bold 12px/30px Dotum, sans-serif }
6 text-align 텍스트의 정렬 방향
  • left : 왼쪽 정렬
  • right : 오른쪽 정렬
  • center : 중앙 정렬
  • justify : 양쪽 정렬
 text-align: right;
text-align: left;
text-align: center;
text-align: justify;
7 background 태그의 배경 지정
  • background-color : 배경 색
  • background-image : 배경 이미지 - background-image: url("이미지 경로")
  • background-repeat : 배경 이미지 반복 여부 - no-repeat, repeat-x, repeat-y, repeat
  • background-position : 배경 이미지 위치
 background: #ffffff url('image.png') no-repeat 10px center;
8 border 태그의 테두리 설정
  • border-width : 테두리의 두께
  • border-style : 테두리의 스타일 - solid, dotted, dashed, double, groove, ridge, inset, outset
  • border-color : 테두리의 색상
 border-width: 2px;
border-color: #000000;
9 border-radius 태그의 테두리를 둥글게 만들어 줌. border 속성 없이도 사용 가능. px단위와 % 사용가능 border-radius: 5px;
border-radius: 1px 2px 3px 4px;
10 visibility 태그의 가시성 결정
  • visible : 보임
  • hidden : 숨김 (자신의 영역은 계속 차지)
  • collapse : 겹치도록 지정
  • inherit : 부모 요소의 값을 상속
 visibility: visible;
visibility: hidden;
visibility: collapse;
visibility: inherit;
